# Movie Releases Bot

Bluesky bot for theatrical + digital movie release announcements, powered by TMDB API and GitHub Actions.

## BMAD

BMAD methodology installed. Agents and tools in `_bmad/`, artifacts in `_bmad-output/`.
Then invoke agents via /bmad-architect, /bmad-create-architecture, /bmad-create-product-brief, etc.
